# 🐍 Snake Game (HTML / CSS / JavaScript)

브라우저에서 실행되는 **스네이크 게임(Snake Game)** 프로젝트입니다.  
하나의 HTML 파일(`snake.html`) 안에 **HTML + CSS + JavaScript**가 모두 포함되어 있으며, 별도의 설치나 빌드 과정 없이 실행됩니다.

---

## 🎮 특징 (Features)

- 단일 HTML 파일 구성  
- Canvas 기반 렌더링  
- 키보드 화살표 키(← ↑ → ↓) 조작  
- 먹이를 먹으면 길이 증가 & 점수 증가  
- 점수 5점마다 **자동 속도 증가**  
- 벽 또는 자기 몸과 충돌 시 게임 오버  
- 최고 점수 **localStorage 저장**  
- PC / 모바일 브라우저 모두 실행 가능
